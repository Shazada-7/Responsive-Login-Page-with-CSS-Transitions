# 🚀 Responsive Login Page with CSS Transitions

A clean, modern, and fully responsive login page built with HTML5 and CSS3. This project focuses on a "mobile-first" approach and smooth user experience through CSS transitions.

![Giff-7767-ezgif com-resize](https://github.com/user-attachments/assets/4d252ec6-3840-42fb-8fd0-79a0e71c27e6)


## 📱 Features
* **Fully Responsive:** Adapts seamlessly to mobile, tablet, and desktop screens.
* **Smooth Transitions:** Interactive hover effects and focus states on input fields.
* **Modern UI:** Minimalist design using Flexbox/Grid.
* **Form Validation Ready:** Includes HTML5 validation for email and password fields.

## 🛠️ Built With
* **HTML5:** Semantic structure.
* **CSS3:** Custom properties (variables), Flexbox, and Transitions.
* **Google Fonts:** ( "Bebas Neue", sans-serif).
